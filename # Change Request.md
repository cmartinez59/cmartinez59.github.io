# Change Request

## Proposed Change
Add an embedded Google Map to my midterm project website that shows my hometown. The map would be placed on my main personal page so visitors can see where I’m from and interact with the map right on the site.

## Reasons for the Change
My instructor probably wants this to make the site feel more personal and to show that I can handle adding outside content into a webpage. It also gives me practice responding to a real change request, which is part of the PMUPM process we’re learning.

## How This Change Will Add Value
The map makes the page more interesting and gives people a better sense of who I am. It adds something visual and interactive instead of just text. It also shows that I can adjust my project when a stakeholder asks for something new, which is a big part of project management.

## How This Change Will Affect the Constraints
- **Time:** It will take a little extra time to get the embed code, place it correctly, and make sure it fits the layout.
- **Quality:** This actually improves the quality because it makes the site look more polished and modern.
- **Budget:** No cost. Google Maps embeds are free.
